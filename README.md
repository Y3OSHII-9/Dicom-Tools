## üìö Using Dicom-Tools

Once the application is running, you will see the main interface. Here are some basic steps to help you get started:

1. **Load DICOM Files:** Use the 'Load' button to import your DICOM files. Dicom-Tools supports a broad range of DICOM formats.
2. **Test Features:** Explore the various testing options available. You can run tests for different DICOM libraries like DCMTK or GDCM directly from the interface.
3. **View Results:** After running tests, results will appear on the screen. You can save these results for later review using the 'Save' option.
